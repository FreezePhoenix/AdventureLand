Valentines Day Event on Adventure Land

Candy Pop's drop globally. You can eat them or exchange them from Xyn in Mainland.

Love Goo spawns every hour in a random map and inside a random monster pack. Love Goo is co-op but has 98% avoidance, so it's a bit tricky to land a hit! :)

Snippet

game.on("event",function(data){ // "pinkgoo" is the keyname of 'Love Goo' if(data.name=="pinkgoo") { smart_move(data); // The data only includes "map" and no coordinates // So you'll need to search for "pinkgoo" in one of the monster spawns } });
