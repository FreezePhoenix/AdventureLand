Giga Crab Beach Event

Giga Crab is a challenging server event boss. As long as there are Huge Crabs around, Giga Crab only receives one damage! The entire server needs to get together to continually clear all the Huge Crabs that spawn. The event lasts 40 minutes but as long as Giga Crab is being engaged, the event goes on!

Giga Crab's ID is "crabxx" - It spawns regular Huge Crab's "crabx"
