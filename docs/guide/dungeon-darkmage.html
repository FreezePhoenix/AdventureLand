Lair of the Dark Mage [Dungeon]

You need the Frozen Key to enter the instanced dungeon, once a party member has unlocked the dungeon, other party members will be able to directly enter.

First phase has a random Freeze Burst every 6 seconds that freezes the opponent for 12 seconds! G.monsters.xmagefz

Second stage has Anger / random targeting and the Multi Burn skill that hurls a burning fireball to everyone! G.monsters.xmagefi

Third stage has Tangle, Reflection and Self Healing. G.monsters.xmagen

The last stage has 12,640 damage, Immunity, Anger and Warp/Stomp! G.monsters.xmagex

Note: Instances are only alive as long as there's someone inside and online!

enter("winter_instance"); // To unlock using a key enter("winter_instance",player.in); // To enter an already unlocked dungeon
