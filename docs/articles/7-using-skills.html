In Code, skills are referenced by their keys from: G.skills

One example is "supershot" you can check out: G.skills.supershot

// Going Invis with a Rogue use_skill("invis"); // Casting Merchant's Luck use_skill("mluck",get_player("LuckyCharacter")); // Using Supershot with a Ranger var target=get_nearest_monster(); if(is_in_range(target,"supershot") && !is_on_cooldown("supershot") && character.mp>=G.skills.supershot.mp) use_skill("supershot",target); // Checks if a target is in "supershot" range, makes sure "supershot" isn't on cooldown and character has enough mana

Reference functions: use_skill is_in_range is_on_cooldown

Special Usage: 3-Shot

var targets=[]; for(id in parent.entities) if(parent.entities[id].type=="monster" && is_in_range(parent.entities[id],"3shot") && targets.length<3) targets.push(parent.entities[id]); // Use 3-Shot with a Ranger on 3 targets use_skill("3shot",targets);

Special Usage: Controlled Burst

var targets=[]; for(id in parent.entities) if(parent.entities[id].type=="monster" && is_in_range(parent.entities[id],"cburst")) targets.push([parent.entities[id],100]); // Attempt to use "Controlled Burst" for 100 mana on all monsters in range using a Mage use_skill("cburst",targets);

Special Usage: Blink

use_skill("blink",[character.x+10,character.y+10]);
