Adventure Land uses Electron. When you stumble onto an issue, you can google the issue by combining error phrases with "electron".

Just Re-Open or Re-Install

Very rarely Steam glitches and the game can't connect to Steamworks, so restarting Steam is a must. Sometimes simply doing a clean install could be the solution.

Steam: Run from the Command Line

Copy paste this to Terminal by inserting your own system user: /Users/YOUR_USER/Library/Application\ Support/Steam/steamapps/common/adventureland/Adventure\ Land.app/Contents/MacOS/Adventure\ Land
You'll likely stumble onto the issue that you can report/search

App Store: Run from the Command Line

Copy paste this to Terminal: /Applications/Adventure\ Land.app/Contents/MacOS/Adventure\ Land

App Store: Can't Install - "Unable to Download App"

Please email hello@adventure.land
